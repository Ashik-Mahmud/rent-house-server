# Rent House App Server

## Description

This is the server for the Rent House App. It is a Node.js server that uses Express.js and MongoDB.

## Technologies Usages

- Node.js
- Express.js
- MongoDB
- Mongoose
- Dotenv
- Nodemon
- Cors
- bycrypt
- jsonwebtoken
- Heroku
- Postman
- Git
